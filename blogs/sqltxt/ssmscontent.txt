 			<p class="pstyle"> SSMS is a powerful tool and below are its benefits:</p>
                        <ul>
                            <li>First and foremost,most people dont like to remember commands,and this tool  provides a graphical interface to manage Sql Server</li>
                            <li>Used for ~configuring~, managing, and administering all components within Microsoft SQL Server</li>
                            <li>Apart from connecting to database engine, it also connects to analysis services, integration services and reporting services.With <b>SSMS 17.2 onwards</b>, the Performance Dashboard now comes as part of the Standard Reports. We do not need to download and install it separately.With this dashboard, its helps in troubleshooting performance issues</li>
                            <li>It also supports managing Azure components such as Azure SQL Database and Data Warehouse, as well as hybrid cloud features such as StretchDB</li>
                            <li>It's a gateway to SQL Profiler</li>
                            <li>Can ~develop~ T-SQL code</li>                            
                        </ul>
