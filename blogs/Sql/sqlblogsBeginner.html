SQL Server Management Studio

You can download SQL Server Manamgent Studio(SSMS) from the official Microsoft Website. SSMS is a powerful tool and below are its benefits:

  • First and foremost,most people dont like to remember commands,and this tool provides a graphical interface to manage Sql Server
  • Used for configuring, managing, and administering all components within Microsoft SQL Server
  • You can open SSMS in Windows by clicking on "Search" and start typing in "Sql...",it should show the SSMS if its installed as shown in the image below
