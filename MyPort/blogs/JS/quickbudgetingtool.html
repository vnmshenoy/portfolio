Budgeting tool

In this blog we will learn how to create a budgeting tool using HTML5, jQuery,jQuery Canvas js and Bootstrap

What we will achieve at the end of this blog

This page has link to the demo and complete source code of the application.You can download the code and use it in your own project(ofcourse needs to be tailored to your requirement) or extend to include your new features like OCR etc. The application allows you to:

  • Setup a budget
  • Displays the expenses,amount remaining etc in form of pie chart(uses jQuery canvas)
  • You can drill down each slice of the pie chart and figure out more details about the items
  • Update the budget
